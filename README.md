# slugcase

A library to convert a string to slug format

## Installation

```sh
npm install slugcase

```

## Usage

```js
import { slugcase } from 'slugcase',

const myString = 'Hola como estas?';

const mySlugString = slugcase(myString);

// mySlugString
// hola-como-estas
```
